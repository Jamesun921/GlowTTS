# Privacy Policy / 隐私政策

Last updated: 2026-02-09

## English

### 1. Overview
GlowTTS is a browser extension that provides text-to-speech features on web pages.

### 2. Data We Collect
GlowTTS does not collect personal identity data for sale or advertising.

For core functionality, GlowTTS may process:
- Extension settings you configure (stored locally via browser extension storage)
- Web page text needed to perform text-to-speech features
- Network request metadata required for extension functionality

### 3. How We Use Data
Data is used only to:
- Provide text-to-speech features
- Save your extension preferences
- Maintain compatibility and reliability of extension functions

### 4. Data Sharing
GlowTTS does not sell user data.
GlowTTS does not share personal data with third parties except when technically required to deliver requested functionality.

### 5. Data Storage and Retention
- Settings are stored in browser extension storage.
- Processed page content is handled as needed for runtime functionality.
- No unnecessary long-term retention is intended.

### 6. Security
Reasonable technical measures are used to reduce unauthorized access and misuse.

### 7. Your Choices
You can:
- Remove the extension at any time
- Clear extension data through browser extension management tools

### 8. Changes to This Policy
This policy may be updated when product behavior or legal requirements change. The latest version is indicated by the "Last updated" date above.

### 9. Contact
For privacy questions, contact: `james@beatuare.com`

## 中文

### 1. 概述
GlowTTS 是一个浏览器扩展，用于在网页中提供文本转语音功能。

### 2. 我们收集的数据
GlowTTS 不会为了出售或广告定向而收集个人身份信息。

为实现核心功能，GlowTTS 可能会处理：
- 你配置的扩展设置（通过浏览器扩展存储在本地）
- 执行文本朗读所需的网页文本内容
- 实现扩展功能所需的网络请求相关元数据

### 3. 数据用途
数据仅用于：
- 提供文本朗读功能
- 保存你的扩展偏好设置
- 保障扩展功能的兼容性与稳定性

### 4. 数据共享
GlowTTS 不出售用户数据。
除实现你请求的功能所必需的技术路径外，不会向第三方共享个人数据。

### 5. 数据存储与保留
- 设置数据存储于浏览器扩展存储中。
- 网页内容仅在运行时按需处理。
- 不进行不必要的长期保存。

### 6. 安全措施
我们采取合理的技术措施，降低未经授权访问或滥用风险。

### 7. 你的选择
你可以：
- 随时卸载扩展
- 通过浏览器扩展管理工具清除扩展数据

### 8. 政策更新
当产品行为或法律要求发生变化时，本政策可能更新。请以本文顶部的“Last updated”日期为准。

### 9. 联系方式
如有隐私相关问题，请联系：`james@beatuare.com`
